    Sobre o Projeto

    Existem muitas fontes de informação no mundo atual. Esta informação é armazenada das mais variads formas: documentos na web, imagens, sons, diagramas, mapas, tabelas e banco de dados de variados graus de complexidade. O objetivo principal deste projeto é o estudo de metodologias para integração de dados provenientes de fontes heterogêneas (banco de dados, web, etc.), de maneira a identificar mecanismos que possam ser aplicados ao processo de identificação e integração de informações semanticamente equivalentes, armazenadas em meios estruturados e/ou semi-estruturados.
